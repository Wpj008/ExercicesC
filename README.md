# Exercices en langage C

Ce dépôt contient une série d’exercices écrits en langage C visant à pratiquer les bases de la programmation.

## Contenu
Les exercices couvrent notamment :
- Déclaration et utilisation des variables
- Opérations et calculs
- Structures conditionnelles (if, else, switch)
- Boucles (for, while, do while)
- Fonctions
- Tableaux
- Chaines de caractères

## Objectif
Renforcer la compréhension des concepts fondamentaux du langage C à travers des exercices pratiques.
